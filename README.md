# spark-frontend
Admin dashboard for Spark 


https://user-images.githubusercontent.com/70111426/215267221-13dde637-728c-4722-aab2-7d0f47de11b9.mp4



## Installation

```bash
$ npm install
```

## Running the app

```bash
# development mode
$ npm run dev

# production mode
$ npm run build
```

## Test

Run both tests and code coverage
```bash
$ npm run test
```

Run only tests
```bash
$ npm run mocha
```

## EsLint
    
```bash
# lint
$ npm run eslint

# lint fix
$ npm run fix
```
