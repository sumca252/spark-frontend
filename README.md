## Installation

```bash
$ npm install
```

## Running the app

```bash
# development mode
$ npm run dev

# production mode
$ npm run build
```

## Test

Run both tests and code coverage
```bash
$ npm run test
```

Run only tests
```bash
$ npm run mocha
```

## EsLint
    
```bash
# lint
$ npm run eslint

# lint fix
$ npm run fix
```
