# spark-frontend

Admin dashboard for Spark 

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development mode
$ npm run dev

# production mode
$ npm run build
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```

## EsLint
    
```bash
# lint
$ npm run eslint

# lint fix
$ npm run fix
```
